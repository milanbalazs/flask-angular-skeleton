# Skeleton for web application

## Tech

### Backend
 - Python (Flask [RESTful])

### Frontend
 - TypeScript (Angular)

### Webserver
 - Nginx

### Deployment
 - Docker (Docker Compose)

## Deploy

### Run the following command:

```bash
docker-compose up --build
```

## Accesses

### Backend (RESTful API)

```text
http://localhost:9099/api/<endpoint>
```

**Example:**

```text
http://localhost:9099/api/ping
```

### Frontend

```text
http://localhost:9099
```
